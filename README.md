# personsrest

  #### Develop a backend with Spring Boot where there are 4 endpoint. 
    /rest/pessoas        (List all persons)
    /rest/pessoa/{id}  (Find a person by ID)
    /rest/pessoa/save/ (Save a Person). 
    /rest/pessoa/remove/{id}  (Delete a person by ID)
    
 #### A person has:
    o	Name
    o	ID
    o	Street
    o	Number
    o	Neighborhood
    o	City
    o	State
    o	Cellphone
    o	Phone
